Age of Empire II; The Conquerors Unit ID List

This is Age of Empire II; TC Unit ID List which CAT translated research result by Tempura and AOK AI Colosseum UnitID research team.

If you want to count a UnitID of red color name, you must use (cc-players-unit-type-count 0 <unitID> <rel-op> <value>). 0 is a player number of gaia.

You have the freedom to reproduce, disseminate and publish this content. If you find out some mistakes in this document or find new Unit ID, send me e-mail please.

Special thanks : Elite Raider, Bingmann, Tempura, Yoan, Jawa, Sekkou
